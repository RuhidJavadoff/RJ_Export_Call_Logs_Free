# üìû Call Logs Export App

## üåü About the Project

"Call Logs Export App" is an innovative mobile application designed for business companies, enabling easy recording of calls and generation of detailed reports. The app helps you manage and analyze incoming and outgoing calls, and export them in various formats: **.txt**, **.xlsx** (Excel), and **.pdf**.

Prioritizing user convenience, the application offers **dark and light mode** options for a visual experience tailored to your preference. To serve a wide audience, the app supports multiple languages, including: **Azerbaijani**, **English**, **Turkish**, **Russian**, **Swahili**, **Akan**, **Shona**, **Northern Ndebele**, **Filipino**, **Sinhala**, **Tamil**, **Hindi**, **Korean**, and **Japanese**.
